O objetivo deste projeto é conhecer o perfil dos clientes da Telco que desistiram dos serviços da empresa.
Os problemas a serem resolvidos são: 
    Qual o perfil dos clientes que deixaram de utilizar os serviços da empresa?
    Quanto a empresa deixa de faturar por este motivo?
Neste projeto somente uma Análise Exploratória de Dados (EDA) foi suficiente para obter-se a resposta para
os problemas, obtendo-se as características, ou perfil, dos clientes que abandonam a empresa.
Utilizaou-se neste projeto o conjunto de dados da empresa Telco, que está disponível no Kaggle com o 
título Telco Customer Churn, no link: https://www.kaggle.com/dataset/blastchar/telco-customer-churn
